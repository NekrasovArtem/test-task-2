# Тестовое задание 2

Тестовое задание для отбора на производственную практику

## Технологии
- Gulp
- PUG
- SASS
- JS

## Использование 
Установите npm-пакет с помощью команды:
```sh
npm i
``` 

## Разработка

##### Запуск версии для разработки:
```sh
gulp
```

##### Сборка исходный файлов:
```sh
gulp build
```

##### Сборка pug-файлов:
```sh
gulp puged
```

##### Сборка стилей:
```sh
gulp styles
```

##### Сборка скриптов:
```sh
gulp scripts
```

##### Сборка изображений:

```sh
gulp images
```

##### Очистка директории dist
```sh
gulp reset
```
